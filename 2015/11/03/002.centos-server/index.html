正好实验室有空闲电脑，就自己搭一个玩儿啦～。
在纠结半天，看了好多资(si)料(bi)后，决定选用 CentOS 作为操作系统， 主要开发环境为 Python 和 Node.js。

  • 下载 Python 3.5.0 ，之后解压，安装…
